# Document State

## Tujuan

Mendefinisikan status AcademicDocument selama proses berlangsung.

---

## State

```text
NEW
```

Dokumen baru dibuat.

---

```text
INITIALIZED
```

Metadata telah diisi.

---

```text
TEMPLATE_LOADED
```

Template telah diterapkan.

---

```text
COMPOSED
```

Seluruh Block telah tersusun.

---

```text
FORMATTED
```

Format dokumen telah diterapkan.

---

```text
VALIDATED
```

Validasi selesai.

---

```text
RENDERED
```

Dokumen siap diekspor.

---

```text
EXPORTED
```

Dokumen berhasil disimpan.
